Vidi-menuen er hovedmenuen i kortapplikationen og findes typisk i venstre side af skærmen. Her får du adgang til alle de centrale funktioner til administration af lag, søgning, tegning, print og meget mere.

![Vidi-menuen](../resources/vidi-menu-vidi-vidi-menu-oversigt.svg)
*Vidi-menuen placeret til venstre i kortvinduet*

:::tip[Godt at vide]
Vidi-menuens indhold kan variere afhængig af den valgte konfiguration og aktiverede udvidelser.
:::

## Tilgængelige funktioner

### Lagstyring
Administrer hvilke kortlag der vises på kortet. Du kan:
- Tænde og slukke for lag og laggrupper
- Søge i lag-data
- Justere gennemsigtighed
- Vise labels (etiketter)
- Filtrere data med avancerede filtre
- Downloade data i forskellige formater
- Omrokere lag og laggrupper
- Se metadata og information om lag

[Læs mere om lagstyring →](lagstyring)

### Infoværktøj
Hent detaljeret information om features på kortet:
- Klik på features for at se attributter
- Redigér features (kræver Editor-udvidelse og rettigheder)
- Avanceret søgning med buffer og tegneværktøjer
- Se info på features inden for et område
- Attributtabel med resultater

[Læs mere om infoværktøj →](infovaerktoj)

### Søgning
Søg efter adresser, matrikler, ejendomsnumre og brugerdefineret data:
- Adresser i Danmark
- Matrikelnumre
- Ejendomsnumre (ESR)
- Brugerdefinerede søgninger (f.eks. lednings-ID, knudenavne)

[Læs mere om søgning →](soegning)

### Baggrundskort
Vælg og administrer baggrundskort:
- Vælg mellem forskellige baggrundskort
- Juster gennemsigtighed
- Se info om baggrundskort
- Sammenlign baggrundskort med vertikal slider (split-screen)
- Blend baggrundskort med horisontal slider (overlay)

[Læs mere om baggrundskort →](baggrundskort)

### Signaturforklaring
Vis forklaring af symboler og farver på kortet:
- Vis signaturforklaring for aktive lag
- Vis signaturforklaring ovenpå kortet
- Deaktiver lag fra signaturforklaring
- Automatisk opdatering ved lagændringer

[Læs mere om signaturforklaring →](signaturforklaring)

### Tegneværktøj
Tegn direkte på kortet:
- Linjer med forskellige stilarter
- Arealer/polygoner
- Rektangler og cirkler
- Punkter/markører
- Tekst-annotationer
- Redigér og slet tegninger
- Vis mål på linjer (valgfrit)

[Læs mere om tegneværktøj →](tegnevaerktoj)

### Projekter
Gem og del kortopsætninger:
- Gem kortets tilstand (lag, zoom, tegninger, m.m.)
- Indlæs gemte projekter
- Del projekter via link
- Lokalt eller bruger-gemt
- Omdøb og slet projekter

[Læs mere om projekter →](projekter)

### Print
Eksportér kort til PDF eller PNG:
- Vælg papirstørrelse og orientering
- Tilføj titel og kommentar
- Inkluder signaturforklaring
- Download eller vis i browser
- HTML-version til web-integration

[Læs mere om print →](print)

## Udvidelser i menuen

Afhængigt af konfigurationen kan følgende udvidelser (extensions) også være tilgængelige i Vidi-menuen:

### Konfliksøgning
Find overlappende features mellem lag. Generer PDF- og Excel-rapporter.

[Læs mere om konfliksøgning →](/vidi/extensions/konfliksoegning)

### Koordinater
Arbejd med koordinater i forskellige systemer (WGS84, ETRS89 UTM32N). Se og "gå til" koordinater.

[Læs mere om koordinater →](/vidi/extensions/koordinater)

### Skrå- og gadefoto
Integrer med Google Street View, Mapillary, Skråfoto og andre billedtjenester.

[Læs mere om skrå- og gadefoto →](/vidi/extensions/skraa-gadefoto)

### Symboler
Placer SVG-symboler på kortet med skalering og rotation.

[Læs mere om symboler →](/vidi/extensions/symboler)
